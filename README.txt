# CONSIDERACIONES SOBRE LA CREACIÓN DEL CURSO

1. Trata de evitar errores en la ficha económica, esta debe ser la primera que debes rellenas.
    - Revisa el código
    - Revisa las fechas
    - Pon en observaciones si es funcionario GVA o NO y el lugar de realización
    - Recuerda que debes convertir la FICHA económica a pdf y mantén los dos archivos, respeta los nombres
    - Cualquier error por mínimo que sea en la ficha económica tiene difícil solución ya que se abren muchos procedimientos a partir de la misma
    - Recuerda enviar por portafirmas el archivo

2. Si cambias el nombre de algún archivo este queda fuera del índice de búsqueda (Es como si no existiera)

3. Si existen varios ponentes copia tantas veces como sea necesario el archivo DATOS PONENTE_NOMBRE.pdf y las autorizaciones asociadas.

4. Cuando termine la formación debes cambiar el nombre de la carpeta del curso poniéndole e sufijo _TERMINADO

5. Borra este archivo si tienes claras todas las indicaciones

6. La carpeta CODIGO-Tec incluye documentación que debes rellenar, a esta debes añadir:

    - CODIGO_ACTA FINAL.pdf: Acta final del curso con todos los participantes y logos de FSE (por ejemplo 25FP99CF999_ACTA FINAL.pdf)
    - CODIGO_ACTA PONENTE.pdf: Acta que firma el ponente con los APTOS y no APTOS
    - CODIGO_AULES.pdf: Si es un curso de Aules y sacas las actividades realizadas de Aules
